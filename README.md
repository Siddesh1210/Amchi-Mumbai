# Amchi-Mumbai
This project is all about for tourist who tend to visit popular places in Mumbai in short period of time. 
The website consists of most attracted places in Mumbai and best hotels, waterpark, market, malls and temples to visit in Mumbai.
website link : https://amchi-mumbai.vercel.app/
